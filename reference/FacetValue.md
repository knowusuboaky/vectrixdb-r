# Facet Value

A single facet value with count

## Public fields

- `value`:

  The facet value

- `count`:

  Number of occurrences

## Methods

### Public methods

- [`FacetValue$new()`](#method-FacetValue-new)

- [`FacetValue$clone()`](#method-FacetValue-clone)

------------------------------------------------------------------------

### Method `new()`

Create a new FacetValue

#### Usage

    FacetValue$new(value, count)

#### Arguments

- `value`:

  The value

- `count`:

  The count

------------------------------------------------------------------------

### Method `clone()`

The objects of this class are cloneable with this method.

#### Usage

    FacetValue$clone(deep = FALSE)

#### Arguments

- `deep`:

  Whether to make a deep clone.
