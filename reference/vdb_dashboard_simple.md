# Launch Simple Dashboard

Convenience wrapper around
[`vdb_dashboard()`](https://knowusuboaky.github.io/vectrixdb-r/reference/vdb_dashboard.md)
using `db$path`.

## Usage

``` r
vdb_dashboard_simple(db)
```

## Arguments

- db:

  Vectrix object.

## Value

Invisibly returns server object from
[`vdb_dashboard()`](https://knowusuboaky.github.io/vectrixdb-r/reference/vdb_dashboard.md).
