# Import from File

Import documents from text file

## Usage

``` r
vdb_import(db, path, separator = "\n")
```

## Arguments

- db:

  Vectrix object or collection name

- path:

  Input file path

- separator:

  Line separator for documents

## Value

Vectrix object

## Examples

``` r
if (FALSE) { # \dontrun{
vdb_import(db, "documents.txt")
} # }
```
