# Reciprocal Rank Fusion (RRF)

Reciprocal Rank Fusion (RRF)

## Usage

``` r
rrf_fusion(rankings, k = 60, weights = NULL)
```

## Arguments

- rankings:

  List of ranked ID vectors

- k:

  RRF constant (default 60)

- weights:

  Optional weights for each ranking

## Value

Named vector of fused scores
