# Delete Collection

Delete a collection

## Usage

``` r
vdb_delete(name, data_dir = NULL, confirm = TRUE)
```

## Arguments

- name:

  Collection name

- data_dir:

  Data directory

- confirm:

  Require confirmation

## Value

Logical success

## Examples

``` r
if (FALSE) { # \dontrun{
vdb_delete("my_docs")
} # }
```
